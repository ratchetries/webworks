What is HTTP? aka hyper text protocol. clients are able to send data to and from a server What is a URL? aka uniform resource locator. an address for an internet resource What is DNS? aka domain name system. takes layman-readable URLs and converts them into an IP
What is a query string? allows users to pass key-value pairs into the URL? ex: ?key1=value1&key2=... List two HTTP Verbs and their use cases. GET - get some data from the server (pages, search forms) POST - send data to the server (pages that change data on server) What is an HTTP request? request from client to server that follows HTTP protocol What is an HTTP response? response from server to client that follows HTTP protocol What is an HTTP header? Give a couple examples of request and response headers you have seen. it provides info about the request or response ex: request headers - accept, cache-control, host, cookie, user-agent response headers - last-modified, set-cookie, cache-control, content-type
